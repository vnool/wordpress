AppStack Theme Documentation

How to quickly setup and use this theme.

Installation & Plugins

In order to install the theme, you have two options:

  1. In the downloaded .zip file go in the /theme folder and upload the AppStack sub-folder contained in it to http://yourdomain.com/wp-content/themes/ directory via any FTP program.

  2. Take the .zip found in the theme folder and install it through the dashboard.
      In order to activate the Theme:
    • Log into your Wordpress admin using the following URL: http://yourdomain.com/ wp-login.php
    • Next, go to Appearance > Themes. Here, you should see a thumbnail of AppStack theme.
    • Click on the Activate link to activate this theme

You will be promted in the WordPress Dashboard to install theme required plugins. Follow the prompts to install them.


Required Plugins

  • ACF Pro

Optional (but Recommended) Plugins

  • Contact Form 7

Demo Content (optional)

Demo content has been included in your download to make it easier to set up the theme. It is of course completely optional.

Go to Tools/Import and install the WordPress importer plugin. You can then use it to upload the included appstack-demo.xml file.


Note: - Make sure you have installed the required plugins before importing the demo content.

Customizing Theme

This theme features advanced customization features with live preview. Click on 'appearance/customize' to open the customizer.

Colors

Highlight colors of the theme can be set.

Logos

You can upload a header logo.

Header Image

Set a sitewide custom header image. The header image is shown on pages using the 'default' template

Note: - You can overide the header image on individual pages (featured image).

Social Links

Add your social links. Any fields with values entered will automatically display the social icon in the footer on the frontend.

Menu Setup

This theme has two menu areas. The main menu which lives at the top of pages (primary menu). And a footer menu.

To Create a Menu

  1. Go to Appearance > Menus
  2. Add Menu Name
  3. Click Create Menu
  4. Then select it from Manage Locations
  5. Save
  6. Select the pages that you want in the menus
  7. Add to Menu
  8. Re-arrange them
  9. Save menu

Assign Menus

  1. Go to Appearance > Customize
  2. Under the 'Navigation' tab, Select your Menus

Single Page Links

If using a single page design you can set up section links as follows.

Create a Custom Link

  1. Select the 'custom link' option.
  2. Enter a label & add it to your menu.

Enter Section ID

  1. Enter the section ID preceded by a hashtag.
  2. Note: It should match the ID you set for that section in the page builder.

Note: - The theme will automatically add the proper url to that section when you are on a different page.

Set Home & Blog Page

    Homepage

  1. Create a new page using the homepage template (If you imported the demo content, you can skip this step)
  2. Go to 'appearance/customize' and select the 'Static Front Page' tab. Set the 'Front page'.

    Blog Page

  1. Create a new page using the default template and enter your title/subtitle. Leave the rest of the page blank.
  2. Go to 'appearance/customize' and select the 'Static Front Page' tab. Set the 'Posts page'.


Note: - To set the same page as your homepage and main posts page, simply set the "Posts Page" to your desired page and leave the "Front Page" selection as the default -select-.

Sidebar & Widgets

You can add widgets to the sidebar in Appearance > Widgets. In posts you can select to display the sidebar or leave the page full width.

Posts

The AppStack theme supports several post formats. You can select 'gallery', 'quote', 'audio', 'video' or standard post format from the raio list on the right.

When you click the desired post format the post entry fields will change.

Gallery Format

This format adds an image slider to your post.

Video Format

Adds video above your post. Simply enter the video link. oEmbed does the rest.

Standard Format

This is a regular post. You can anter a 'featured image' to display above the post.

Page Builder

This theme includes simple 'page builder' sections which allow you to quickly layout pages.

When you create a new page or post you will notice the 'layout' section. Simply click on the 'add row' button from the bottom right to pull up the sections menu. Simply select the section you would like to add.


Note: - The 'editor'section pulls up the main WordPress content editor.

Note: - Drag & Drop sections to reorder them. Or hover over the right edge to delete them.

Note: - The 'Parallax Header' section should only be placed first (at the top of a page) & will replace the page header..

Custom Headers

You can add custom header images to pages/posts by uploading a 'featured image'. If no featured image is added the page will use the default header image set in appearance/customize/header image.

Note: - You can hide the header section on a page by checking the 'Hide Header' box.

Note: - header images are best at 1900px + wide.

Social Links

Add your social links in appearance/customize under the 'social' tab. Any links you have entered will be automatically added to the site footer.


Customizing CSS

You can create a child theme to add custom css. See more here.

Icon Set

The theme includes fontawesome icons which can be used in some page builder sections.
